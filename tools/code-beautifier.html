❓ 常见问题

支持哪些编程语言？ +

目前支持JavaScript、CSS、HTML、XML、SQL、JSON等主流编程语言，我们会持续添加更多语言支持。

代码会被上传到服务器吗？ +

不会。所有代码处理都在您的浏览器本地完成，我们不会收集或存储您的任何代码内容，确保您的代码安全。

有代码长度限制吗？ +

理论上没有严格限制，但为了保证处理速度和浏览器性能，建议单次处理的代码不超过10MB。

如何处理语法错误的代码？ +

工具会尽力格式化有语法错误的代码，但可能无法完美处理。建议先修复明显的语法错误再进行美化。

可以自定义格式化规则吗？ +

当前版本使用标准的格式化规则，未来版本会考虑添加自定义配置选项，如缩进大小、换行风格等。
