使用说明

  1. 上传文件：点击上传区域或直接拖拽PDF、Word、Excel文件到页面中
  2. 设置密码：输入您想要设置的密码，建议使用8位以上的强密码
  3. 确认密码：再次输入密码以确保准确性
  4. 开始加密：点击"开始加密"按钮，系统将为您的文件添加密码保护
  5. 下载文件：加密完成后，系统会自动下载加密后的文件

支持的文件格式：

  • PDF文件 (.pdf) - 使用zip.js库创建AES-256加密压缩包
  • Word文档 (.doc, .docx) - 使用zip.js库创建AES-256加密压缩包
  • Excel表格 (.xls, .xlsx) - 使用zip.js库创建AES-256加密压缩包

安全提示：

  • 所有文件处理均在您的浏览器本地完成，不会上传到服务器
  • 请妥善保管您设置的密码，忘记密码将无法打开文件
  • 建议使用包含大小写字母、数字和特殊字符的强密码
  • 加密后的文件大小可能会略有增加

常见问题解答

文件加密是否安全？

是的，我们使用专业的zip.js库创建AES-256加密的ZIP压缩包，这是目前最安全的加密标准之一。所有加密过程都在您的浏览器本地完成，文件不会上传到任何服务器，确保您的隐私和数据安全。

忘记密码怎么办？

很抱歉，如果忘记密码，将无法恢复加密的文件。这是为了确保文件的安全性。建议您将密码记录在安全的地方，或使用密码管理器来管理您的密码。

支持批量加密吗？

是的，您可以同时上传多个文件进行批量加密。所有文件将使用相同的密码进行加密，加密完成后会分别下载每个加密后的文件。

加密后的文件如何打开？

所有文件都会被打包成使用zip.js库创建的AES-256加密ZIP文件。使用时需要先解压ZIP文件（输入密码），然后就可以正常打开原始文件。这种方式确保了文件的完整性和安全性。

文件大小有限制吗？

为了确保良好的用户体验，建议单个文件大小不超过50MB。如果文件过大，加密过程可能会比较缓慢，请耐心等待。

是否需要安装软件？

不需要安装任何软件。这是一个基于Web的在线工具，只需要现代浏览器即可使用。推荐使用Chrome、Firefox、Safari或Edge浏览器以获得最佳体验。
