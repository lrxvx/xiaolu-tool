📖 使用指南

  1. 选择HTTP请求方法（GET、POST、HEAD等）
  2. 输入要测试的API接口地址
  3. 在"请求头"选项卡中配置必要的HTTP头信息
  4. 在"请求体"选项卡中输入POST/PUT请求的数据（支持JSON格式）
  5. 点击"发送请求"按钮执行API调用
  6. 查看响应结果，支持JSON自动格式化和复制功能

🔧 工具详细介绍

简易版Postman是一款专业的在线API接口测试工具，为开发者提供便捷的接口调试解决方案。无需安装任何软件，直接在浏览器中即可完成各种HTTP请求的测试和调试工作。

多种请求方法

❓ 常见问题

支持哪些HTTP请求方法？ +

支持GET、POST、HEAD、PUT、DELETE、PATCH等常用HTTP请求方法，基本涵盖了RESTful API的所有需求。

如何处理需要认证的API？ +

在"请求头"选项卡中添加Authorization头，例如：Bearer your-token-here，或者添加API Key等认证信息。

为什么有些请求会失败？ +

可能是由于CORS跨域限制。某些API服务器不允许浏览器直接访问，这是浏览器的安全限制，不是工具问题。

支持文件上传吗？ +

当前版本主要支持JSON数据的发送，文件上传功能将在后续版本中添加。

请求数据会被保存吗？ +

不会。所有请求都是直接从您的浏览器发送到目标服务器，我们不会保存任何请求数据或响应内容。
